# HVN Anti-AI UI

React UI patterns for landing pages, dashboards, and modular interfaces that avoid generic AI-generated frontend aesthetics.

HVN Anti-AI UI is the owner for anti-AI UI guidance that used to live as scattered ORCA or HVN Stack checklist text. ORCA should route here; HVN Stack should list it as a distinct capability.

Use the docs site for live examples:

```sh
npm run dev
```

The library is intentionally opinionated. It starts from content hierarchy, asymmetry, human-tuned rhythm, and concrete copy.

See `docs/orca-hvn-transition.mdx` for the repo boundary.
