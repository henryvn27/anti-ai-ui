# Design Philosophy

Start from content and hierarchy, not from a generic grid. A page should reveal what matters most before it tries to look balanced.

## Principles

- Break perfect symmetry in controlled ways.
- Use distinctive typography and irregular type steps.
- Avoid default palettes and generic gradients.
- Use custom easing curves that feel hand-tuned.
- Let motion explain state or narrative, not decorate.
- Use non-linear spacing where content pressure demands it.
- Design around real content and edge cases.

## What Generic AI Would Do Instead

Generic AI frontend output tends to use a centered hero, broad promise, gradient background, three feature cards, icon circles, and uniform rounded panels. That structure is easy to generate and hard to remember.

## What This Library Does

The components create controlled imbalance: offset features, lopsided metrics, diagonal timelines, tilted panels, and copy primitives that accept specific language instead of filler.
