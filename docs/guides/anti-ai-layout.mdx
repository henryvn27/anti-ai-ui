# Anti-AI Layout

Generic layouts make every unit equal. Real interfaces usually have hierarchy, awkward data, long labels, urgent states, and secondary actions.

Use:

- Unequal columns.
- Offset rows.
- Big-small-negative-space compositions.
- Irregular gaps.
- Asymmetry tied to content importance.

Avoid:

- Card grids where every item has the same size.
- Decorative asymmetry that makes scanning harder.
- Breaking conventions in transaction-heavy flows where standard controls are safer.
