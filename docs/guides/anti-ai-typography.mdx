# Anti-AI Typography

Generic typography is too even. Every heading step feels mathematically clean, every paragraph has the same rhythm, and every label looks like a default utility class.

This library uses:

- A display serif paired with a compact sans.
- Slightly irregular type steps.
- Tight display leading.
- Mono labels for operational details.
- Mood presets for quiet, dense, loud, and editorial text.

The goal is not chaos. The goal is a type system with judgment.
