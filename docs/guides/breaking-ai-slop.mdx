# Breaking AI Slop

Use this checklist before shipping a page.

## Design

- Does this page rely on a generic three-card feature row?
- Is the palette a default purple-blue gradient system?
- Is everything perfectly symmetric?
- Are icons placed in circles everywhere?
- Is the most important content visually dominant?
- Would the layout still make sense if AI tools disappeared tomorrow?

## Writing

- Are words like "powerful," "seamless," or "next-generation" unsupported?
- Are there concrete details, stakes, or examples?
- Does the wording sound like someone at this company would say it out loud?
- Could the copy be pasted onto a different product without changing much?
